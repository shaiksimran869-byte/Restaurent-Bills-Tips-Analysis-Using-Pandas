Average tip by gender

  • A bar plot is used when we want to compare categories.
  • You can quickly see whether Males tip more,Females tip more, or Both tip equall
  • x-axis = category (sex -> Male/Female)
  • y-axis = numeric value (tip)
  • This is called categorical vs numeric comparison
  • When you want to compare a numeric value across categories, bar plot is the best option.
In[23]:
sns.barplot(data=df,x='tip',y='sex')
plt.title("Average Tip By Gender")
plt.show()

Average Tip by Gender (Bar Plot)

This bar plot shows the average tip amount given by male and female customers. The x-axis represents the average tip, and the y-axis shows the gender categories. From the plot, we can observe:

  • Male customers tip slightly more on average than female customers.
  • The bar length represents the mean tip amount.
  • The small horizontal black line on each bar is the confidence interval, which shows the variation in the tip values. This visualization makes it easy to compare tipping behavior between different genders.
